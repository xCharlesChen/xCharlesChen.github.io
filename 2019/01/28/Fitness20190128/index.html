基本的健身知识，包括热量计算和饮食摄入计划，健身三分练，七分靠饮食。

三大营养素

  • 碳水

    直接为身体提供能量。增肌多吃，减脂少吃，靠脂肪来提供能量。减脂可以深入慢速吸收的复合型碳水（红薯、玉米、糙米、燕麦、白米饭），少吃精白面，拒绝糖果、饮料、果汁。

  • 蛋白质

    合成肌肉，增肌原料，减脂能减少过量肌肉分解。

  • 脂肪

    也是提供肌肉，但是不是直接，相对缓慢，要避免所有的垃圾脂肪。可摄入一定量的优质脂肪有（不饱和多元脂肪酸）：肉类、鸡蛋黄、坚果、橄榄油、水果（牛油果之类的）。

热量计算公式

单位参考:
weight(kg) height(cm) age(years old) time(minute) Q(cal,卡路里)

男生日常消耗 (*就是乘法) ：

Q1 = 66.5 + 13.75 * weight + 5.003 * height - 6.775 * age

女生日常消耗：

Q1 = 65.1 + 9.563 * weight + 1.85 * height - 4.676 * age

运动消耗通式：

Q2 = 0.086 * weight * time

增肌摄入

IMQ = Q1 + Q2 + 500

减脂

DEQ = Q1 + Q2 - 500

计算你的特殊 Q 值

以我为例，体重 73 kg,裸高 178 cm， 24岁，代入男生日常消耗公式计算 Q1：
Q1 = 66.5 + 13.75 * 73 + 5.003 * 178 - 6.775 * 24 = 1798.184 也就是 1800 卡。
Q2 = 0.086 * 73 * 90 = 565 cal
IMQ = 1800 + 565 + 500 = 2865 cal
DEQ = 1800 + 565 - 500 = 1865 cal

三大营养素摄入热量比例：
碳水：蛋白质：脂肪 = 3： 4 : 3
按增肌算
碳水： 2865 * 0.3 = 858.5 cal -> 214 g
蛋白质： 2865 * 0.4 = 1146 cal -> 286 g
脂肪： 2865 * 0.3 = 858.5 cal -> 95g

上面换算最终质量参考公式

1g 碳水提供 4 cal
1g 蛋白质提供 4 cal
1g 脂肪提供 9cal

按照上面换算结果也就是我要摄入 214g 的碳水，286g 的蛋白质以及 95g 的脂肪。摄入碳水、蛋白质、脂肪的可数都已经计算出来，你可以在手机上下载 薄荷app 来计算你所摄入的物质含有多少 g 碳水、蛋白质、脂肪。
举例：
碳水有紫薯、山药、糙米。
薄荷app 中食物估算 100g 的紫薯有 15g 碳水。
一个蛋白 3 g 蛋白质，整蛋 9 g 蛋白质。
100g 瘦肉有 20 g 蛋白质，高标准肉算的了。减脂不吃 猪肉、鸭肉、羊肉。好的肉：牛肉、鱼肉、虾肉、无皮鸡肉。
脂肪靠坚果摄入，买小包装那种每日一包，上面有写的很清楚的营养成分。

减脂建议

减脂期间最好再早餐摄入大量碳水来提供你这一天需要的能量。如果你一天要吃三个红薯，建议早餐一个，练前练后一小时各一个。好让让碳水都消耗掉，不然如果你在办公室坐着的时候摄入大量碳水，都转换成肝糖原肌糖原甚至脂肪这种非期望产物了。意思就是尽量把大量碳水安排在早餐和你训练前后，为了让你消耗掉它。

不管你是增肌还是减脂，训练后必须要吃东西。不然就徒劳辛苦了，让你的付出收益最大化。
