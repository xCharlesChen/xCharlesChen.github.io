从零开始搭建你的个人网站！

1.购买域名

阿里云链接

阿里云注册账号购买即可。
如果你需要个性化自己的个人网站才需要购买。如果不需要请跳过第一步，你可以搭载在 github 上。
下面举例最终效果：

个人
https://xcharleschen.com/

github搭载
https://xcharleschen.github.io/chengl.github.io/

2.下载 git

git 下载链接
git 操作指南
git ssh 设置

下载完成后,需要连接到你的 git 仓库，详细参考 git 操作指南。
以下简述。
命令行输入以下命令: 

git config --global user.name NAME
git config --global user.email EMAIL
ssh-keygen -t rsa -C EMAIL

NAME 是你的 github 账号， EMAIL 是你的注册 github 所使用邮箱。
第三行命令是用来生成 ssh key，生成的 id_rsa.pub 需要复制其内容到上面的 github ssh 设置链接中 new ssh。

3.下载 Node.js

Node 下载链接

1.下载安装后在命令行中输入以下命令进行版本验证 node 安装成功 

node -v

我的命令行展示结果为：

v10.15.0

2.国内因为某些不能说的原因导致安装东西很慢的，我们需要设置一个淘宝镜像为后面的所有 npm 加速。
设置 cnpm,以后你的所有 npm 操作都用 cnpm 代替 npm 关键词。 

npm install -g cnpm --registry=https://registry.npm.taobao.org

例如你要 npm 某个包，直接用 cnpm 代替 npm： 

cnpm install 某个包名

4.建立 Hexo 项目

1.安装 hexo : 

cnpm install -g hexo

2.类似确认 node.js 的安装版本验证以下 hexo 是否正常安装： 

hexo -v

3.初始化一个 hexo 项目，你需要建立一个文件夹用来保存你的项目，最好按照 hexo 手册定义名字
为 你的github账号名.github.io,我的就叫 chengl.github.io，虽然我的 git 账号是 xCharlesChen。
在这个目录下右键选择 Git Bush Here,输入以下代码进行初始化并安装依赖: 

hexo init
hexo install

4.修改 _config.yml 配置如下，你需要以下面的代码为模板，改下 repo 参数(参数是你的 github 上
的仓库地址，仓库名最好也第二步中的文件夹名，拷贝过来加个 .git 结尾)，其他两个可以先别管。 

deploy:
  type: git
  repo: https://github.com/xCharlesChen/chengl.github.io.git
  branch: master

5.本地项目文件夹中右键选择 Git Bush Here,输入以下代码（第三步的命令行交互没关就接着用那个交互窗口）： 

hexo clean
hexo generate
hexo server

本地预览： http://localhost:4000
6.发布到网站：
部署插件安装

cnpm install hexo-deployer-git --save

发布到网站 

hexo clean
hexo generate
hexo deploy

5.使用个人域名

1.第四步中的 6.发布到网站只会挂载到 github 上面，如果你需要个性化，进入你的阿里云控制台，解析你的域名。解析设置：

记录类型选择 CNAME；
主机记录这里可以不填，如果你加入 www,这样你以后访问个人网站需要在你购买的域名前加 www；
解析线路 默认；
记录值：你的仓库包名，也就是上面那个 repo 配置中的最后一段，例如我的： chengl.github.io;
TTL 默认；
确认保存。

2.在你的本地项目中的 /resource 下添加一个文件类型为所有文件的文件，在文件类输入你的个人域名，例如我的：xcharleschen.com
修改了需要发布，执行以下代码。 

hexo clean
hexo generate
hexo deploy

6.github 设置你的个性化地址

进入你的 hexo项目仓库中，进入 settings，往下拉，找到 custom domain 配置，输入你的域名，例如我的：
xcharleschen.com
保存。
