Markdown 语法归纳，这里记录并举例了大部分常用的语法，基本涵盖了所有的例子。

标题：
# 定级,N个 # 代表 hN
#h1级标题
##h2级标题
###h3级标题
####h4级标题
#####h5级标题
######h6级标题

H1，H2还可以通过 = ，- 定级最高阶和第二阶，例如
This is an H1
=============

This is an H2
-------------


分割线：三个以上的星号/减号/底线 即可作出分割线
****
----
————————


超链接：
[连接名称](网址 , 标题)
[链接名](https://xcharleschen.com, "标题")
[<i class="icon-refresh"></i> 点我刷新](/sonfilename/)

还有一种超链接写法：
[链接名][链接代号]
[link][001]
然后在别的地方定义 001 这个详细链接信息，
[001]: https://xcharleschen.com "个人博客"

直接展示链接的写法：<https://xcharleschen.com>


强调：
用 * _ 包围起来
*斜体强调*
**粗体强调**
_斜体强调_
__粗体强调__


图片
![Alt text](https://xcharleschen.com/logo.png "blog title")


使用 icon 图标文字
<i class="icon-cog"></i>


段落：以一个空行开始，以一个空行结束，中间的就是一个段落。


无序列表：使用 - + * 三个中的一个再加一个空格 ' '

- 无序列表1
- 无序列表2
- 无序列表3

有序列表：使用 数字 加一个英文句点

1. 有序列表
2. 有序列表
3. 有序列表
4. 有序列表
5. 有序列表


块引用
>给引用的文本开始位置都加一个 '>'，
>便可组成一个块引用。在块引用中，可以结合
>其他markdown元素一块使用，比如 列表。
>**强调**
也可以只在第一行加大于号，其他位置不加。

>- 块引用里使用列表，需要和上面的内容隔开一个空行
>- 必须带一个空格再开始书写。例如一个水果列表：
>- Apple
>- Banana


code格式：反引号
Use the `printf()` function.

``There is a literal backtick (`) here.针对在代码区段内插入反引号的情况`` 

换行缩进形成代码区块

    这里先换行，然后缩进4个空格，之后的内容便可以原样显示了，适合用于显示代码内容。直到文本结束或最后一个存在缩进的行为止。    


表格：

Item     | Value
-------- | ---
apple    | $2
banana   | $1
orange   | $1


键盘键
<kbd>Ctrl+[</kbd> and <kbd>Ctrl+]</kbd>

反斜杠
也就是转义符，比如你不想要让 *abc* 起强调作用，就可以用反斜杠处理下。
\*abc*\
以下符号都可以通过反斜杠来正常展示：
\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   惊叹号
