如何设置 DNS。

为什么要手动设置 DNS ?

一般来说，你选择的宽带运营商会提供一个 DNS 服务器给你，大部分情况下都会给你分配最近的服务器，但是也会出现一些问题比如：

  1. 访问某些网站会很慢，例如 Apple icloud，有的时候甚至所有网站都打不开了。
  2. 如果你输入网址错误，自动设置 DNS 会给你打开一个都是广告的网站。（运营商的额外收入）
  3. 安全问题，设置官方的 DNS 能保证你安全地访问网站。

应该怎么设置最优 DNS 组合？

设置 DNS 方法参考(链接)

对于国内使用，我的建议是首选 DNS 服务器设置为国内的 114.114.114.114， 备用 DNS 服务器设置为 Google 的 8.8.8.8。

这样当你在访问一些国外的网站的时候，如果首选 DNS 服务器解析很慢，会自动用备用 DNS 服务器进行快速解析，一般来说 Google 的够用了。
