如何优雅的使用布局。

简要介绍之前的居中方法

前端工程师一般通常需要做的一个布局就是水平、垂直居中。CSS 中的 float 就是用来干这个的，但是它有很多弊端，比如影响父级元素。刚接触遇到居中的问题的时候，你肯定会想能不能像 word 中一样使用 float: center。然而 CSS 并不支持这样做。你又想到了 CSS 中的 text-align: center 或是 verticle-align: center，但是它们确只能作用在行内元素上，对块级元素无效。

还记得当年网页布局中都是通过 table 元素来实现的，在 table 的单元格中使用 align/valign 实现水平和垂直的居中。但是，Web 偏向语义化了，当初这种折中的方式被业内取缔，CSS 出现了新的 3 种布局方式，包括：标准文档流、浮动布局以及定位布局。

回忆下简单的水平居中： 

margin: 0 auto

水平垂直居中：

.parent {
    position: relative;
}
.kids {
    position: absolute;
    margin: auto;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
}

或者

.parent {
    position: relative;
}
.kids {
    width: 100px;
    height: 100px;
    position: absolute;
    top: 50%;
    left: 50%;
    margin-top: -50px;
    margin-left: -50px;
}

这些方式都太麻烦了，特别是第二种，明显的对适配是不友善的。下面我们引出我们的 flex 布局。

flex 布局基本概念

首先要明白 flex 布局的几个基本概念，轴和容器。

  • 轴
    1. 主轴 main axis
      1. 主轴方向 flex-direction: row / column / row-reverse / column-reverse
    2. 交叉轴 cross axis
  • 容器
    1. 父容器 container
      1. 水平方向属性 justify-content: center / flex-start / flex-end / space-around / space-between
      2. 竖直方向属性 align-items: center / flex-start / flex-end / baseline / stretch
    2. 子容器 item
      1. 占据比例 flex: 见参考图
      2. 单个 item 交叉轴排列 align-self: center / flex-start / flex-end / baseline / stretch

下面举例一个子元素垂直居中，对于父元素，首先要使用 display: flex，然后用 justify-content: center 实现水平居中， align-items: center 实现垂直居中。

.dad {
    display: flex;
    justify-content: center; // 水平居中
    align-items: center; // 垂直居中
}
