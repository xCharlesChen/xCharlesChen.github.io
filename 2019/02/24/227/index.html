Javascript String 字符串相关方法归纳。

注意，本文中没有声明 str 的地方，都是默认为：

let str = 'Hello, Buddy!'

  • 指定脚标字符方法

  1. charAt(index)

    参数：

    index: 以 0 为起始位的 Number 类型脚标。

    作用：

    返回该脚标的字符所组成的字符串。

  2. charCodeAt(index)

    参数：

    index: 以 0 为起始位的 Number 类型脚标。

    作用：

    返回该脚标的字符的字符编码。

    let str = 'Hello, Buddy!';
console.log(str.charAt(7));
//  B
console.log(str.charCodeAt(7));
//  66

  • 字符串操作方法

  1. concat(str1, str2)

    参数：

    str1,str2...一个或者多个字符串。

    作用：

    该函数返回一个被创建的新的字符串,不影响原字符串。

    let str = 'Hello, Buddy!';
//  "Hello, Buddy!"
str.concat('111', '222')
//  "Hello, Buddy!111222"
str
//  "Hello, Buddy!"

  2. substring(firstIndex, lastIndex)/slice(firstIndex, lastIndex)

    参数：

    firstIndex: 新字符串起始位置。

    lastIndex: 新字符串结束位置, 但是不包含该脚标的字符。如果该值小于 firstIndex ,则反向截取。

    作用：

    该函数返回一个被创建的新的字符串,不影响原字符串。

    str.substring(2, 6)
//  "llo,"
str
//  "Hello, Buddy!"

  1. substr(firstIndex, number)

    参数：

    firstIndex: 新字符串起始位置。

    number: 新字符串的长度。

    作用：

    该函数返回一个被创建的长度为 number 新的字符串,不影响原字符串。

    str.substr(2, 6);
//  "llo, B"
str
//  "Hello, Buddy!"

  • 字符位置查询方法

  1. indexOf(str)

  2. lastIndexOf(str)

    参数：

    str： 目标查找字符段。

    作用：

    indexOf() 从 0 开始查询，lastIndexOf() 从字符串末尾开始查询，返回匹配字符串的脚标位置。

    str.indexOf('d');
9
str.lastIndexOf('d');
10

  • 字符串大小写切换

  1. toLowerCase()
  2. toLocaleLowerCase()
  3. toUpperCase()

  4. toLocaleUpperCase()

    str.toLowerCase()
//  "hello, buddy!"
str.toLocaleLowerCase()
//  "hello, buddy!"
str.toUpperCase()
//  "HELLO, BUDDY!"
str.toLocaleUpperCase()
//  "HELLO, BUDDY!"

  • ES6 新增

  1. repeat(times)

    参数：

    times: number 类型，重复的次数。

    作用：

    该函数返回一个被创建的重复拼接次数为 times 新的字符串,不影响原字符串。

    str.repeat(2)
//  "Hello, Buddy!Hello, Buddy!"

  2. includes(str)

    参数：

    str: 目标查询字符串。

    作用：

    字符串是否包含目标字符串，返回布尔值。

  1. startWith(str)

    参数：

    str: 目标查询字符串。

    作用：

    字符串是否以目标字符串为起始，返回布尔值。

  2. endsWith()

    参数：

    str: 目标查询字符串。

    作用：

    字符串是否以目标字符串结束，返回布尔值。

  • 去空格

  1. trim()

    参数：

    无参数。

    作用：

    去掉字符串左边的空格。

    let str2 = '    2  3  abc ';
console.log(str2);
console.log(str2.trim());
console.log(str2);
//    2  3  abc 
//2  3  abc
//    2  3  abc

    扩展：

    下面用正则表达式扩展各种去空格方法。

    /*
方法 trim 去掉空白
参数 [str]    要处理的字符串        
参数 [type]   
    类型：
        l 去除左边的空白    
        r去除右边空白    
        b去掉两边的空白        
        a去除所有空白
*/
function trim (str,type) {
    var type=type||"b";
    if(type=="b"){
        return str.replace(/^\s*|\s*$/g,"");
    }else if(type=="l"){
        return str.replace(/^\s*/g,"");
    }else if(type=="r"){
        return str.replace(/\s*$/g,"");
    }else if(type=="a"){
        return str.replace(/\s*/g,"");
    }
}

  • 字符串的模式匹配

  1. match(pattern)

    参数：

    pattern: 一个正则表达式或 RegExp 对象。

    作用：

    返回一的数组，里面包含被插叙到的字段。

    let str = 'cat, hat, bat';
let pattern = /.at/;
let pattern2 = /.at/g;
let matches = str.match(pattern);
console.log(matches);
//  如果正则不是全局检测，则查找单个，并且返回单个的详细信息。
//  ["cat", index: 0, input: "cat, hat, bat", groups: undefined]
let matches2 = str.match(pattern2);
console.log(matches2);
//  ["cat", "hat", "bat"]

  1. search(pattern)

    参数：

    pattern: 一个正则表达式或 RegExp 对象。

    作用：

    返回字符串中第一个匹配项的索引，如果没有找到，则返回-1

    let str = 'cat, hat, bat';
str.search('hat');
//  5
str.search('boss');
//  -1

  1. replace(arguments[0], arguments[1])

    参数：

    arguments[0]: 一个RegExp对象或者一个字符串。

    arguments[1]: 一个字符串或一个函数。

    作用：

    利用replace()进行替换的时候，

    1. 如果传入的是字符串，则只会替换第一个子字符串，要想替换所有的子字符串，则需要传入一个正则表达式，而且要指定全局（g）标志。不影响原来的字符串，也是生成一个新的字符串。

      let text = 'cat, hat, bat';
let result = text.replace('at', 'ost');
console.log(result);
//  cost, hat, bat
result = text.replace(/at/g, 'ost');
console.log(result);
//  cost, host, bost

    2. 如果传入的是一个函数。第二个参数为函数时函数的返回值作为替换字符串。那就是说与第二个参数是字符串一样，如果第一个参数是正则表达式，并且全局匹配，则这个函数的方法将被多次调用，每次匹配都会被调用。

      function replacer(match , p1 , p2 , p3 , offset , string){
    // p1 is nondigits, p2 digits, and p3 non-alphanumerics
    console.log(`${match}
                 ${p1}
                 ${p2}
                 ${p3}
                 ${offset}
                 ${string}`); 
    /* => abc12345#$*%
         abc
         12345
         #$*%
         0
         abc12345#$*%"    */         
    console.log([p1, p2, p3].join(' - ')); // => "abc - 12345 - #$*%"
    return [p1, p2, p3].join(' - ');
}
var newString = 'abc12345#$*%'.replace(/([^\d]*)(\d*)([^\w]*)/, replacer); // =>"abc - 12345 - #$*%"

  • 与数组之间的转换

  1. split(str, num)

    参数：

    str: 用于指定字符串位置切割。

    num：切割后的数组长度。

    作用：

    把字符串中包含的某些字符作为分界点切割该字符串组成数组。

    let text = 'cat, hat, bat';
text.split(',');
//  ["cat", " hat", " bat"]
text.split(',', 2);
//  ["cat", " hat"]
text.split(/[^\,]+/);
//  ["", ",", ",", ""]

  • 比较两个字符串

  1. localeCompare(str)

    参数：

    str：要比较的字符串。

    作用：

    从第一个位置开始,按字母表比较，被比较的字符串只要每一位都在 str 参数的每一位之前，则返回 -1，反之返回 1, 如果相等则返回 0。

    let str1 = 'aaabb';
let str2 = 'cdac';
let str3 = 'acac';
str1.localeCompare(str2);
//  -1
str1.localeCompare(str3);
//  -1
str2 = 'aaaaa';
str1.localeCompare(str2);
//  1
str2 = 'aaaba';
str1.localeCompare(str2);
//  1
str2 = 'aaabc';
str1.localeCompare(str2);
//  -1

  • String 构造字符串

  1. fromCharCode(num)

    参数：

    num: 一个或者多个字符编码。

    作用：

    将接收到的一个或多个字符串编码转换成一个字符串，这个方法与实例方法charCodeAt()执行相反的操作。

    let str = 'Ace';
str.charCodeAt(0);
//  65
String.fromCharCode(65);
//  "A"
